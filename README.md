# 3SplitWebcam
simple 3 split video recorder using OpenCV

### Features
- Show Current Webcam video using cv.VideoCapture
- Save webcam video by avi file using cv.VideoWriter
- Preview and Record Mode
- press Spacebar to record, esc to quit

### Additional features
- When recording the video splits into 3 areas(just like when streamers and bj do reaction in Korea)
- codec: FourCC
- Image Crop, Image Flip(mirror effect)
- hconcat
